# Search Algorithms & Intelligent Pathfinding System

AI search and pathfinding system built with Python implementing BFS, DFS, A* search, heuristic optimization, and graph traversal algorithms in interactive environments.

This project focuses on graph search algorithms, heuristic optimization, intelligent agent navigation, and classical AI problem-solving techniques.

---

## Features

- Breadth-First Search (BFS)
- Depth-First Search (DFS)
- A* Search implementation
- Uniform Cost Search (UCS)
- Greedy Best-First Search
- Heuristic function optimization
- Graph traversal and shortest-path analysis
- Intelligent navigation in text-based environments

---
